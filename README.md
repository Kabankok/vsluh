# Вслух

**Выделил текст — услышал.** Локальная озвучка выделенного текста живым
русским голосом. Работает офлайн, ничего не уходит в интернет, бесплатно.

</div>

---

## Что это

Выделяете любой текст в любом окне (статья, письмо, чат, PDF), жмёте
**Ctrl + Alt + Z** — и слышите его голосом. Отпустить уши от экрана: слушать
статьи, вычитывать свои тексты на слух, разбирать длинные переписки, пока руки
заняты другим.

- 🔊 **6 русских голосов** на выбор — мужские и женские.
- 💻 **Полностью офлайн.** Голос синтезируется на вашем компьютере. Ни одна
  буква не уходит в интернет — приватно.
- ⚡ **Быстро.** Начинает читать примерно через секунду. Есть видеокарта NVIDIA —
  использует её; нет — работает на процессоре.
- 🎚 **Настройки в окне:** голос, скорость, свой словарь произношений
  (чтобы бренды, латиница и артикулы читались правильно).
- 🆓 **Бесплатно и с чистой лицензией** (MIT) — можно пользоваться как угодно.

Сделано для селлеров маркетплейсов и всех, кто много читает с экрана.

## Установка

> Нужен установленный **Python 3.10–3.12** ([python.org](https://www.python.org/downloads/),
> при установке поставьте галочку «Add Python to PATH»). Папку держите в пути
> без русских букв (например `C:\vsluh`).

1. Скачайте проект: кнопка **Code → Download ZIP**, распакуйте.
2. Запустите **`install.bat`** двойным кликом.
   Он поставит всё нужное, скачает голос (~90 МБ) и пропишет автозапуск.
3. Готово. Возле часов появится иконка 🔊.

## Как пользоваться

| Действие | Клавиши |
|---|---|
| Озвучить выделенное | **Ctrl + Alt + Z** |
| Остановить | **Ctrl + Alt + X** |

- **Настройки** — правый клик по иконке в трее → «Настройки…» (или ярлык
  «Вслух — настройки» на рабочем столе). Там: выбор голоса с кнопкой
  «прослушать», скорость, словарь произношений, свои горячие клавиши.
- **Выключить/включить** озвучку и сменить голос — прямо из меню в трее.
- **Выход** — из меню в трее.

## Портативность

Вся программа лежит в одной папке (голос, настройки — рядом). Хотите перенести
на другой компьютер — скопируйте папку и запустите `install.bat` там ещё раз
(он до-настроит окружение под новую машину).

## Как это устроено (для любопытных)

Два процесса: **движок** (модель Silero + ударения RUAccent + горячие клавиши +
локальный сервер настроек) и лёгкий **трей** (иконка, общается с движком по
`127.0.0.1`). Так torch и оконный интерфейс не конфликтуют. Захват выделения —
нативный `Ctrl+C` + чтение буфера через WinAPI, без сторонних хук-библиотек.
